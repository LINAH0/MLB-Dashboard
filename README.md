📦 공유 파일
-----------
MLB_Dashboard_주차별 입고현황.html  (단일 파일, 모든 기능 포함)

🚀 빠른 시작 (3단계)
-------------------
1. MLB_Dashboard_배포용.html 파일을 더블클릭하여 브라우저에서 실행
2. 상단의 "파일 선택" 버튼을 클릭하여 엑셀 파일 업로드
3. 자동으로 데이터가 로드되고 차트/테이블이 표시됩니다

💻 시스템 요구사항
-----------------
- 운영체제: Windows, macOS, Linux (모든 OS 지원)
- 웹 브라우저: Chrome, Edge, Firefox, Safari (최신 버전 권장)
- 인터넷 연결: 최초 실행 시 차트 라이브러리 로드를 위해 필요

📊 주요 기능
-----------
- 실시간 KPI 대시보드 (누적 발주, 입고, 달성률)
- 제품별/국가별/협력사별 필터링
- 주차별 추이 차트 및 상세 테이블
- DELAY 이슈 자동 감지 및 표시

⚠️ 주의사항
----------
- 모든 데이터는 로컬에서만 처리되며 서버로 전송되지 않습니다
- 최초 실행 시 인터넷 연결이 필요합니다 (차트 라이브러리 다운로드)
- 엑셀 파일은 .xlsx 형식을 지원합니다
- 엑셀 파일에 "생산현황" 시트가 있어야 합니다 (다른 시트는 무시됨)
